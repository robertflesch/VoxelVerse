Classpublic class AbstractMockDispatcher
InheritanceAbstractMockDispatcher flash.events.EventDispatcher
ImplementsIMockDispatcher

Class provides a base for all mock dispatchers to build on top of. Override both dispatch and cancel to create the mock data that is required by the given request. For the dispatch method, after constructing the data calling super.dispatch will cause the dispatcher to fire a DispatcherEvent.RESULT in an Asynchronous manner via setTimeout. Set the delay property to change the default delay of setTimeout,which is 500 milliseconds.
