Interfacepublic interface IMockDispatcher extends flash.events.IEventDispatcher
ImplementorsAbstractMockDispatcher

Interface defines the API for a mock dispatcher. The responsibility of a mock dispatcher is to fake a dispatcher to a given service and return data in the same format that the request's parser is expecting.
