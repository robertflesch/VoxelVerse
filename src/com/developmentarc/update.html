Updates

Package Renaming

DevelopmentArc Core has renamed its package structure from com.developmentarc.framework to com.developmentarc.core. There has also been some re-organization of the various micro-architectures into seperate packages. Going forward each new system (micro-architecture) will be organized in a similar manner. Below is a breakdown of each change.

  • Command and Action Delegation - Has been organized under the com.developmentarc.core.actions package
  • Task System - Is now located in com.developmentarc.core.tasks package
  • Service Request DelegationNEW - Is located under com.developmentarc.core.services package

LocalConnectionManager

  • Added a disconnect() method to disconnect from the current session and removes all event handlers. This addition is to address bug #1
