DevelopmentArc TM Core 0.09 Notes

Introduction

DevelopmentArc is proud to release a new version of it's Core framework. This version is one step closer to our goal of a 1.0 release. This release adds a large new component (Service Request Delegation System) to our suite of systems (micro-architectures) as well as a much needed refactor of the package structure of the library and other assorted bug fixes.

License

DevelopmentArcTM is released under the MIT License.

Download

DevelopmentArc TM Core .09 can be downloaded from Google Code here. We have released a SWC library to be added directly into your project libs directory. We also provide a ZIP file which contains the SWC, source code and ASDoc generated documentation.

Online Source

A read-only version of the Core source can be checked out using Subversion. Developers can tie directly into the .09 branch at http://developmentarc-core.googlecode.com/svn/branches/core_v0.9/. HEAD is available but is the location of the newest check-ins from the DevelopmentArc team and we do not guarantee the code is production ready, stable or corresponds with the available documentation.
